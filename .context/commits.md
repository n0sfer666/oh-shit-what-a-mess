# Коммиты

- Conventional commits, английский: `feat(scope): ...`, `fix(scope): ...`, `chore: ...`, `test: ...`.
- Scope = крейт: `core`, `tui`, `cli` (или без scope для общего).
- Без AI-меток (Co-Authored-By, Generated by и т.п.).
- Коммит только на зелёных проверках (`.claude/.verify-state.json`, hook-энфорсмент).
- Ветки: `feature/...`, `fix/...` от основной. Прямой коммит в main запрещён.
